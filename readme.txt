== Description ==

Плагин для WordPress "Pokazania IPU - Передача показаний счётчиков ЖКХ" - для передачи показаний индивидуальных приборов учёта без регистрации и "Личного кабинета". 
Демо плагина здесь: https://kodabra.unchikov.ru/pokazaniya-ipu-demo/.
Плагин создавался и используется для передачи показаний ИПУ по расходу горячей воды в квартирах. Рабочий вариант можно посмотреть здесь: https://teplo28.ru/pokazania/.
Данный плагин не использует базу SQL для хранения показаний - весь массив данных хранится в файле, который необходимо обновлять 1 раз в месяц.

В плагине можно настроить:

1. Количество знаков в лицевом счёте (По умолчанию 12);
2. День месяца начала приёма показаний ИПУ (По умолчанию 15);	
3. День месяца окончания приёма показаний ИПУ (По умолчанию 25);	
4. Email для приёма показаний ИПУ (По умолчанию Email Администратора);
5. Заголовок формы (Передать показания приборов учета горячей воды);
6. УРЛ-ссылка на логотип для размещения в сообщении клиенту о принятии показаний.

Для работы плагина необходимо каждый месяц выгружать из Вашей учетной программы (1С:Предприятие) данные с показаниями в следующем виде (каждый Лицевой счет с данными с новой строки):
Лицевой счет;Улица;Адрес;Счетчик1;Показание1;Счетчик2;Показание2;Счетчик3;Показание3;
Пример:

000000027008;Чехова;г. Белогорск, Чехова ул., 39А, кв. 68;Счетчик Чех39А68/к;3.000000;Счетчик Чех39А-68В;0.113000;;;
000000033823;Вольный;г. Белогорск, Вольный пер., 7, кв. 53;СчетВольн 7-53/в;0;СчетВольн 7-53/к;0;СчетВольн 7-53/с;0;
000000004247;Кирова;г. Белогорск, Кирова ул., 247/2, кв. 81;Счеткирова 247/2/81;4.294000;;;;;

Копировать их, вставлять в текстовое поле на странице "Настройки Показания ИПУ" и сохранять.

Максимальное количество счётчиков в одном Лицевом счёте - 3.


== Installation ==

1. Загрузите плагин "Pokazania IPU - Передача показаний счётчиков ЖКХ" на сайт WordPress и активируйте его. 

2. Перейдите на страницу Админки Настройки -> Показания ИПУ. Заполните и сохраните настройки.

3. Вставьте в текстовое поле данные с показаниями из Вашей учетной программы (1С:Предприятие) в следующем порядке (каждый Лицевой счет с данными с новой строки):
Лицевой счет;Улица;Адрес;Счетчик1;Показание1;Счетчик2;Показание2;Счетчик3;Показание3;
Пример:

000000027008;Чехова;г. Белогорск, Чехова ул., 39А, кв. 68;Счетчик Чех39А68/к;3.000000;Счетчик Чех39А-68В;0.113000;;;
000000033823;Вольный;г. Белогорск, Вольный пер., 7, кв. 53;СчетВольн 7-53/в;0;СчетВольн 7-53/к;0;СчетВольн 7-53/с;0;
000000004247;Кирова;г. Белогорск, Кирова ул., 247/2, кв. 81;Счеткирова 247/2/81;4.294000;;;;;

Нажмите "Сохранить показания ИПУ".

4. Разместите шорткод [un_pokazaniya_ipu] на странице передачи показаний.
